📋 Descripción General

Guaca Analytics Pro es una aplicación web moderna desarrollada con React y Vite. El proyecto está diseñado para proporcionar análisis de datos, predicciones y capacidades de OCR con una interfaz intuitiva centrada en animales y estadísticas.

🏗️ Estructura por Carpetas

📦 public/

Activos estáticos públicos

  • favicon.png - Icono del sitio
  • logo.png - Logo de la app
  • manifest.json - PWA manifest

🎨 components/

Componentes React reutilizables

  • Header - Encabezado
  • BottomMenu - Menú inferior
  • Card - Tarjeta genérica
  • AnimalCard - Tarjeta de animales
  • TripletaCard - Tarjeta triplete
  • Countdown - Contador regresivo

📄 pages/

Páginas principales de la aplicación

  • Dashboard - Panel principal
  • Historial - Histórico de datos
  • Estadísticas - Análisis
  • Predictor - Predicciones
  • OCR - Reconocimiento óptico
  • Configuración - Ajustes

⚙️ services/

Servicios y utilidades

  • api.js - Llamadas API
  • analytics.js - Análisis
  • ocr.js - OCR integration

📊 data/

Datos estáticos

  • animales.json - Base de animales
